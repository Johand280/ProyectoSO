Componentes Implementados

COMPONENTE 01
Virtualización y Particionamiento Manual

Instalación y conexión mediante Red Interna en VirtualBox. Configuración de direccionamiento estático e ingreso administrativo por SSH.

COMPONENTE 02
Contenedorización con Docker

Construcción de imágenes personalizadas en Docker a partir de Rocky Linux, enlazando servicios web mediante mapeo de puertos dinámicos.

services:
  frontend:
    build: ./frontend
    ports:
      - "80:80"
  backend:
    build: ./backend
    ports:
      - "5000:5000"
COMPONENTE 03
Orquestación Elástica con Kubernetes

Migración al clúster local de Minikube. Control de réplicas concurrentes usando archivos declarativos de configuración YAML expuestos con NodePort.
